# React Native Website Application

تطبيق موقع إلكتروني شامل مطور باستخدام React Native يوفر تجربة مستخدم متكاملة عبر منصات متعددة.

## المزايا الرئيسية

- **واجهة مستخدم حديثة**: تصميم عصري وجذاب يدعم اللغة العربية
- **تنقل سلس**: نظام تنقل سهل الاستخدام بين الصفحات
- **تجاوب كامل**: يعمل بشكل مثالي على جميع أحجام الشاشات
- **محتوى شامل**: يتضمن صفحات الرئيسية، حول الموقع، الخدمات، والتواصل
- **تفاعل المستخدم**: إمكانية إرسال الرسائل والتواصل المباشر

## الصفحات المتاحة

### 🏠 الصفحة الرئيسية
- ترحيب بالزوار
- عرض المزايا الرئيسية
- أزرار التنقل السريع
- معلومات عن الخدمات

### ℹ️ حول الموقع
- معلومات عن الشركة
- الرؤية والرسالة
- القيم والمبادئ
- تاريخ الشركة

### 🛠️ الخدمات
- تطوير تطبيقات الهاتف
- تطوير المواقع الإلكترونية
- تصميم واجهات المستخدم
- الاستشارات التقنية
- الدعم والصيانة

### 📞 تواصل معنا
- معلومات التواصل
- نموذج إرسال الرسائل
- ساعات العمل
- روابط التواصل المباشر

## التقنيات المستخدمة

- **React Native**: إطار العمل الأساسي
- **React Navigation**: للتنقل بين الصفحات
- **React Native Vector Icons**: للأيقونات
- **StyleSheet**: للتنسيق والتصميم

## متطلبات التشغيل

- Node.js 18 أو أحدث
- npm أو yarn
- React Native CLI
- Android Studio (لتطوير Android)
- Xcode (لتطوير iOS)

## كيفية التثبيت

1. استنسخ المستودع:
```bash
git clone https://github.com/KarrarMusawu/React-Native-.git
cd React-Native-
```

2. ثبت المكتبات:
```bash
npm install
```

3. شغل التطبيق على Android:
```bash
npm run android
```

4. شغل التطبيق على iOS:
```bash
npm run ios
```

## البناء والتطوير

### تشغيل Metro Server:
```bash
npm start
```

### فحص الكود:
```bash
npm run lint
```

### تشغيل الاختبارات:
```bash
npm test
```

## هيكل المشروع

```
src/
├── screens/          # صفحات التطبيق
│   ├── HomeScreen.js     # الصفحة الرئيسية
│   ├── AboutScreen.js    # صفحة حول الموقع
│   ├── ServicesScreen.js # صفحة الخدمات
│   └── ContactScreen.js  # صفحة التواصل
├── components/       # المكونات المشتركة
└── styles/          # ملفات التنسيق
```

## التخصيص

يمكنك تخصيص التطبيق عبر:
- تعديل الألوان في ملفات الـ styles
- إضافة صفحات جديدة في مجلد screens
- تحديث معلومات التواصل في ContactScreen.js
- إضافة خدمات جديدة في ServicesScreen.js
